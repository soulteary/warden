# Warden 快速开始示例

本目录包含两个不同复杂度的 Warden 使用示例，帮助你快速上手。

## 📚 示例列表

### 1. [简单示例](./basic/) - 基础使用

**适合场景**：
- 快速测试和开发
- 仅使用本地数据文件
- 学习基本功能

**包含内容**：
- ✅ 本地数据文件配置
- ✅ 基本的 Docker Compose 部署
- ✅ 简单的启动脚本
- ✅ 完整的使用文档

**快速开始**：
```bash
cd basic
docker-compose up -d
```

[查看详细文档 →](./basic/README.md)

### 2. [复杂示例](./advanced/) - 完整功能

**适合场景**：
- 生产环境部署参考
- 需要远程 API 数据源
- 完整的监控和测试

**包含内容**：
- ✅ 本地 + 远程数据源
- ✅ Redis 缓存和分布式锁
- ✅ 定时任务自动同步
- ✅ Mock 远程 API 服务
- ✅ 完整的 Docker Compose 配置
- ✅ 自动化测试脚本
- ✅ 多种数据合并策略演示

**快速开始**：
```bash
cd advanced
cp .env.example .env
docker-compose up -d
```

[查看详细文档 →](./advanced/README.md)

## 🎯 选择指南

### 选择简单示例，如果你：
- 是第一次使用 Warden
- 只需要本地数据文件
- 想要快速验证功能
- 在开发环境中测试

### 选择复杂示例，如果你：
- 需要从远程 API 获取数据
- 需要了解完整的数据合并策略
- 准备部署到生产环境
