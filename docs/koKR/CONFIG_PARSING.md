# 프로그램 매개변수 파싱 설계 문서

> 🌐 **Language / 语言**: [English](../enUS/CONFIG_PARSING.md) | [中文](../zhCN/CONFIG_PARSING.md) | [한국어](CONFIG_PARSING.md)

이 문서는 warden 프로그램의 매개변수 파싱 메커니즘에 대해 자세히 설명합니다. 명령줄 인수, 환경 변수 및 구성 파일 파싱 방법, 우선순위 규칙 및 사용 예제를 포함합니다.

> **참고**: 이 문서는 현재 영어 및 중국어로 제공됩니다. 한국어 버전은 곧 제공될 예정입니다. 당분간은 [영어 버전](../enUS/CONFIG_PARSING.md)을 참조해 주세요.

## 사용 가능한 문서

- [영어 버전](../enUS/CONFIG_PARSING.md) - 완전한 영어 문서
- [중국어 버전](../zhCN/CONFIG_PARSING.md) - 완전한 중국어 문서

## 내용

이 문서는 다음을 다룹니다:

- 아키텍처 설계
- 우선순위 메커니즘
- 지원되는 매개변수
- 사용 방법
- 모범 사례
- 자주 묻는 질문
